		static method setOptions (line 35)
	
 
	
	

Calls the Setter Methods in the given object context for every key  in the supplied options. The Name of the Setter Method must be camelCased  and the key in the $options Array must have underscores  e.g. for the key "file_name" the Setter's name is "setFileName".
