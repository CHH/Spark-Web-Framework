excerpt() - Takes a string and gives back the shortened String.

Looks for the next space within the maximum length plus/minus the tolerance and gives back the string up to this position.

  • return: Shortened string
  • access: public

  • string $string: String to shorten
  • int $maxLength: Maximum length of the shortened string (in characters, approximately)
  • string $ellipsis: Takes the ellipsis which should be appended to the shortened string (default is the typographic ellipsis)
  • int $tol: The space is looked up in the range of $maxLength +/- $tol
